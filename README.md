Deep Learning Model Which Generates The Caption Of The Image.

It is One To Many RNN Model.

Used CNN For Extracting The Features Of The Images, Implemented DesNet201 Pretrained Model To Achieve Better Accuarcy Even With Less Data For Training.

Used LSTM For Generating The Captions in Form Of Sequentail/Textual Data, The Moto Of Using LSTM Is To Get An Accurate Caption Related To Input Image.

Used Google Antigravity For Giving UI To This Deep Learning Model.
